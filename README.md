# LibraryManagementSystem
* Basic application to add,delete the books and student information (issued and returned books).
* steps to use
  * run the .py file
  * username = admin, password = password
  
* **Before you start :** 
  * `pip install python-tk` download tkinter modules
  * `pip install pillow`
